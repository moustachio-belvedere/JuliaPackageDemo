Simple Examples

To begin using JuliaPackageDemo.jl we first need to load in the package with the using syntax. We will also load in PyPlot.jl so we can plot some of the data.

using JuliaPackageDemo
using Plots

Next we will generate a time vector and use this as an input to the function coolfunction1. The result is plotted.

# get time vector
timevec = collect(0:0.1:10)

# get function values at time points
data = coolfunction1.(timevec)

# plot the data
plot(timevec, data)

For more information on coolfunction1 please see the API section. Now we can try something similar with coolfunction2:
